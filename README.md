# Sistema LoRa - Atividade 3

## Descrição
Sistema de comunicação entre dois ESP32 utilizando módulos LoRa RA-02.

## Configuração
- Frequência: 915 MHz
- Comunicação: SPI
- Biblioteca: LoRa.h

## Funcionamento
Um dispositivo envia mensagens numeradas e o outro recebe via Serial Monitor.
